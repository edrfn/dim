Mensagem de consistência de títulos vencidos ao inserir cada item no pedido de venda
*1) Qual o comportamento observado?*

Ao ativar o parâmetro de consistir títulos vencidos em Informações Sobre Vendas e lançar um pedido para um cliente com um título vencido, a mensagem que informa a consistência está sendo exibida a cada item inserido no pedido.

*2) Quais as etapas para reproduzir o comportamento observado?*

Ative o parâmetro para consistir títulos vencidos na venda;
Faça um pedido para um cliente com pedido vencido;
Observe que a mensagem é apresentada a cada produto inserido;


*3) Qual o comportamento esperado?*

Aparecer a mensagem apenas ao escolher o cliente do pedido.

*4) Quais as etapas para executar o comportamento esperado?*
As mesmas etapas do passo 2.
