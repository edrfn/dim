Inutilização de faixa de Nfe - Criação da Interface Visual da Tela de Inutilização de NF-e
*Como:* Usuário do sistema
*Quero:* Construção da interface visual de Inutilização de NF-e
*Para:* Obter uma interface de entrada para os dados referentes a inutilização de NF-e do sistema

*Critério de validação* 
* -Ter acesso à funcionalidade através do menu Vendas->Movimentos-> Inutilização de faixa de NF-e.-
* Atender os requisitos visuais como descrito na especificação #18585.

*Como implementar* 
* Utilizar o novo padrão de telas do sistema;
* Criar um projeto separado para a criação da tela;
* Utilizar-se de telas já feitas no novo padrão como base para a criação da nova tela.
