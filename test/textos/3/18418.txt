Situações encontradas no Parâmetro de Venda Consistência de Títulos Vencidos.
+Obs.: Situação encontrada quando o parâmetro está na opção Permite Liberar+
Estando o parâmetro na opção Permite Liberar, o sistema apenas exibiu uma mensagem com a informação sobre o título do cliente quando deveria, também, exibir a tela para autorizar a venda por meio de uma senha.

*1)Comportamento Observado*
Foi observado que o sistema apenas exibiu uma mensagem com a informação sobre o título do cliente quando deveria, também, exibir a tela para autorizar a venda por meio de uma senha.

*2) Passos para reproduzir o comportamento Observado*
Obs.: Para reproduzir o comportamento é necessário que haja pelo menos um título vencido em nome do cliente para o qual será realizada uma venda.
1- Abrir o sistema e acessar o módulo de vendas;
2- Acessar a tela de Parâmetros e alterar o campo Consistência de Títulos para a opção Permite Liberar;
3- Salvar a alteração, sair do sistema e entrar novamente;
4- Realizar uma venda para o cliente que esteja com título vencido;
5- Observar que o sistema exibe apenas a mensagem de alerta, sendo possível prosseguir com a venda após dar ok na mensagem. 

*3) Comportamento Esperado*
Espera-se que o sistema exiba a mensagem de alerta e também a tela para autorização da venda por meio de uma senha, quando o parâmetro estiver na opção Permite Liberar
